**Análisis de Accidentes de Motocicleta en CR - CA0204**

Material del proyecto individual del curso CA-0204. 
Consiste en un mapa cantonal de Costa Rica que visualiza la frecuencia y gravedad 
de los accidentes de tránsito donde se vieron involucradas motocicletas, facilitando el análisis de datos relevantes sobre este problema de seguridad vial.
